Vigenère Encoder/Decoder

Here you can harness the power of the Vigenère cipher.
Provide the message you wish to encode, as well as the keyword you wish to protect the message with. The message can then be decoded in future as long as the keyword has been remembered.
The app currently accepts a basic lowercase [a-z] alphabet but custom alphabet support is coming shortly.

Haversine Distance Calculator

With this app you can calculate the distance between any two points on the globe using the Haversine Formula. This should be incredibly useful if you are a Discovery Age explorer with Internet access.
Input is taken in degrees, minutes and seconds which are then converted to decimal degrees. Support for direct input of decimal degrees is coming soon.
Here is a list of city co-ordinates to play around with.

The degrees, minutes and seconds should be positive integers within the specified ranges. Any invalid or missing inputs will be set to 0.
